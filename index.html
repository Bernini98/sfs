In the summer of 2020 I started Sites For Students. With the aim of improving the success of university students, graduates and those with minimal work experience, we produce minimal but impactful portfolio websites so opportunity-seekers can display their full range of skills and capabilities to recruiters

CPanel was utilised with a Linux server to run Python web applications and site files

Stripe and its powerful web hooks were implemented for payment processing and start other functions like emailing with regards to subscriptions

Skills in SSH were developed to manipulate files with the Linux server
